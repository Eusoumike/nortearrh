# Nortear Connect

Plataforma operacional B2B para gestão de suporte, onboarding e financeiro do canal Nortear (VR Benefícios + RH Digital).

## Stack

- React 18 + Vite + TypeScript
- Tailwind + shadcn/ui
- Supabase (Postgres + Auth + Storage + Edge Functions)
- React Query
- React Router 6

## Módulos

- Dashboard
- Chamados (kanban, lista, detail com sidebar)
- Onboarding/Implantação
- Clientes
- Financeiro
- Performance
- Base de Conhecimento (Assist)
- Configurações
